KRYVEXIS STARTER DROP

1. Create the app:
   npx create-next-app@latest kryvexis-website --typescript --tailwind --app --src-dir --import-alias "@/*"

2. Go into the project:
   cd kryvexis-website

3. Install animation/icons:
   npm install framer-motion lucide-react

4. Copy the files from this zip into your project root and replace the existing app folder files.

5. Put your logo in:
   public/logo.png

6. Run it:
   npm run dev

7. One-line GitHub push after creating your repo on GitHub:
   git init && git add . && git commit -m "Kryvexis phase 1" && git branch -M main && git remote add origin https://github.com/YOUR_USERNAME/kryvexis-website.git && git push -u origin main
